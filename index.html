Ingeniería en Datos

Presentación

Dr. Julio César Ramírez-Pacheco

Qué hace un ingeniero en datos

  • Recolecta, organiza y limpia los datos para la toma de decisiones.
  • Crear modelos estadísticos avanzados para predecir resultados.
  • Busca patrones ocultos utilizando técnicas de análisis avanzados tales como el aprendizaje automático, inteligencia artificial, etc.
  • Diseñar reportes o tableros para la toma de decisiones en empresas de diversos tipos.
  • Seleccionar técnicas de análisis de datos para diferentes contextos.

¿Cuál es el campo laboral?

  • Arquitecto de datos, científico de datos.
  • Compañías de comercio electrónico.
  • Instancias gubernamentales.
  • Consultor independiente.
  • Director de Business inteligence, gerente de proyectos, gerente de sistemas.

¿Cuál es el perfil de egreso?

  • Haber desarrollado un pensamiento crítico para formular y responder preguntas.
  • Capacidad para la resolución de problemas.
  • Habilidad para abastecer, recopilar, organizar, limpiar, visualizar y modelar datos.
  • Diseñar, desarrollar e implementar hardware para la recolección automática de datos.

¿Cuál es el perfil de egreso?

  • Conocer diversas técnicas de visualización para contar historias con los datos.
  • Capacidad de programar de forma eficiente, soluciones que se adapten a cada empresa.
  • Capacidad de utilizar herramientas matemáticas y estadísticas avanzadas..

¿Cuál es su importancia?

  • Muchos datos se generan de teléfono inteligentes, redes sociales, etc.
  • Identificar maneras de incrementar ganancias en los negocios.
  • Identificar las tendencias en los mercados.
  • Obtener modelos y predicciones útiles en los negocios, salud, etc.
  • Mejorar procesos dentro de una organización basados en los análisis de los datos de la propia empresa
